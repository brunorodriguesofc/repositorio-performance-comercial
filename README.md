📊 Relatório de Performance Comercial em Python

Este projeto simula um sistema simples de análise comercial, avaliando faturamento, impostos e crescimento de vendas ao longo do tempo.

⸻

🚀 Funcionalidades
	•	Cálculo do faturamento total
	•	Cálculo de imposto sobre produtos
	•	Análise de crescimento mensal
	•	Média de crescimento
	•	Classificação de performance comercial

⸻

📌 Regras aplicadas

Impostos
	•	Até 1000 → 10%
	•	Acima de 1000 → 15%

Classificação de performance
	•	≥ 20% → Alta performance
	•	≥ 10% → Média performance
	•	< 10% → Baixa performance

⸻

🧠 Conceitos utilizados
	•	Listas e dicionários
	•	Loop (for)
	•	Estruturas condicionais (if / elif / else)
	•	Cálculo de métricas (crescimento e média)
	•	Formatação de saída

⸻
